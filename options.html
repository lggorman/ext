In order to use a custom set, search quizlet.com for the set you would like to study, and get the set id number from the URL.

For example, if you wanted to learn the Japanese words for animals, you might navigate to the set 'Japanese Animals (romanji)' at the URL http://quizlet.com/30159939/japanese-animals-romaji-flash-cards/. What you need from that URL is the number between quizlet.com and the name of the set. In the case of 'Japanese Animals,' the set id would be 30159939. Happy studying!
